Cliente, Server e Domínio

(Voltar)

Um servidor é um recurso dentro de um sistema computacional, capaz de processar aplicações, armazenar dados e prestar serviços, enquanto o cliente é um computador que solicita esses serviços e recursos. Em uma infraestrutura de TI, ambos formam a arquitetura conhecida como “cliente-servidor”. O cliente somos nós acessando um server, buscando uma pagina ou dados específicos

Um domínio é o endereço de uma pagina na internet, endereço é único e pode ser pago. Toda pagina deve ser hospedada em um server para que seja acessada pelos usuários.

O que é desenvolvimento HTML?

(Voltar)

HTML, sigla para Hypertext Markup Language, é a linguagem de marcação padrão utilizada para estruturar o conteúdo de páginas web. Trata-se de uma tecnologia fundamental no desenvolvimento web, permitindo a criação e formatação de elementos como textos, imagens, links e vídeos

fonte: Mozilla developer

As principais tags

(Voltar)
"Toda tag HTML precisa ser aberta e fechada com os caracteres seguintes: abrir uma tag < >, fechar uma tag < / > "
  • Tags para o corpo de uma pagina: body, head, footer
  • Tag para títulos : h1,h2,h3.
  • Tag para textos : P
  • Tag para destacar um texto em negrito: strong
  • Tag de identificação: ID, class
  • Tag para digitação de textos: input
  • Tag para adição de imagem: img
  • Tag utilizada para adicionar links: a

Para toda imagem adicionada é utilizada uma alt=, alt é um texto que descreve a imagem, para questões de acessibilidade ou até mesmo se por algum erro a imagem não possa ser exibida
